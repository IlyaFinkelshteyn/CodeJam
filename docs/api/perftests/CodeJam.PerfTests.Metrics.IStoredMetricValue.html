    Property Value
    Type Description
    System.Double

    The maximum value. The System.Double.NaN marks the value as unset but updateable during the annotation. The System.Double.PositiveInfinity returned if value is positive infinity (ignored, essentially). IMPORTANT: If the UnitOfMeasurement is not null both Min and Max values are scaled. Use the MetricValueHelpers to normalize them.
