GetMetadataAttributes<TAttribute>(ICustomAttributeProvider)

Performs search for metadata attributes. The search is performed in the following order:

  • member attributes, base implementation attributes (if the attributeProvider is member of the type)
  • type attributes, base type attributes (if the attributeProvider is type or member of the type)
  • container type attributes (if the type is nested type)
  • assembly attributes.
